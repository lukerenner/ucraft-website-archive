# lukerenner.co

Static archive of the Renner Agency site (formerly hosted on Ucraft), now
served for free via GitHub Pages at https://www.lukerenner.co.

- Every page lives at `<clean-url>/index.html` so URLs match the original site.
- `assets/` holds all images, fonts, CSS, and JS, organized by original host.
- `CNAME` binds the custom domain; DNS is managed at iwantmyname.
- The contact form was Ucraft-backed and is currently non-functional.

Built July 2026 from an offline archive plus a fresh crawl of the live site.
